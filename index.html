React

React, is a library built for programming in the JavaScript language specialized for user
interfaces built by Facebook, as it provides a smooth and powerful method for building website interfaces,
as it allows developers to build a component of web pages once so that it can be reused
several times. Wikipedia

Original author: Walke Jordan

Initial release: 7 years ago; May 29, 2013

License: MIT license

Developers: community and Facebook

Written in: JavaScript

React (also known as React.js or ReactJS) is an open-source JavaScript library for building user interfaces or UI components. It is maintained by Facebook and a community of individual developers and companies. React can be used as a base in the development of single-page or mobile applications.
